# CBE346 Core Lab Enzyme Kinetics

This repository contains the reproducible analysis materials for the CBE346 core lab enzyme kinetics report.

## Repository contents

- `export_cbe346_figures.py`
  - Main analysis script used to regenerate report-facing tables and figures.
- `README_NOTEBOOKS.md`
  - Short documentation describing the current analysis workflow and the status of the older notebooks.
- `original_csv/`
  - Raw plate data, calibration data, and legacy velocity table used as inputs to the reproducible analysis.
- `figures_export/`
  - Exported CSV summaries used to support the report discussion, parameter estimates, model comparison, and sensitivity analysis.
- `analysis_summary.md`
  - Short narrative summary of the current inhibition analysis.
